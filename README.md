## Spotify description
Artist [Spotify description](https://open.spotify.com/artist/4M9C2IXUroEUTQRiMKWM46?si=GYXL7md2SJmEUES7qW9fHw) is a sequence of ASCII bytes, that decodes to:
```
The songs whisper nine secrets. Hear them and find the way forward. loopgarden.io/xxxxxxxxx.zip
```

## Discover link - Morse code
3 songs has some Morse code at the end:
1) What She Feels Like ..--- ----. ---.. = 298
2) Iris'Secret --... ..--- ....- = 724
3) Case Closed ---.. ----. ..--- = 892
So the zip file can be downloaded at 
loopgarden.io/298724892.zip  
Check additional documentation in 298724892 folder.
